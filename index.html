HTML

Stands for Hyper Text Markup Language

HTML is made out of:

  • Text content- what you see
  • Mark up- what it looks like
  • Reference to other documents
  • Links to other pages
HTML markup consists of an opening and closing tag and the whole thing is called an element.

This video has a great explanation
Some tags do not need an end tag. the img and break tag do not need an end tag for example.
Thare are also two types of container elements: Blocks which form an invisible box around the content and inline.
